Warehouse-specific features

Operations that don't break

Rotating day/night patterns

Define repeating shift cycles (5 days, 2 off, 5 nights, 3 off) once. Wrok respects them every week automatically.

Role minimums per shift

Set "minimum 1 forklift-certified worker per shift" and "minimum 4 pickers". The scheduler refuses to violate these.

Rest-hour enforcement

Configure minimum 11 hours rest between shifts. Wrok blocks any assignment that would create a violation.

Weekend rotation fairness

The auto-scheduler distributes weekend shifts evenly. Nobody gets stuck with five Saturdays in a row.

Shift manager delegation

Floor supervisors can approve swaps and mark attendance without touching schedule generation. Configurable permissions.

Payroll-ready exports

Excel/CSV export of weekly hours with regular, overtime, night, and weekend columns — ready for payroll software.
