# Payment and account pages are kept out of search by <meta name="robots"
# content="noindex"> on the pages themselves -- deliberately NOT by Disallow.
# A crawler that is disallowed cannot read the noindex tag, and Google may still
# index a blocked URL it has never fetched. Allow the crawl; let noindex work.
#
# Note also: Cloudflare Pages serves extensionless URLs (/payment, not
# /payment.html), so any rule written against the .html paths would match only
# the redirect and never the real page.
